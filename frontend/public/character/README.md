# Character Assets Directory

This folder is designed to store high-quality transparent PNG assets for the interactive virtual mannequin:

## Asset Structure

- `/body.png` - Transparent base mannequin body.
- `/tops/` - Transparent top clothing layers (T-shirts, polo shirts, button-downs).
- `/bottoms/` - Transparent bottom clothing layers (jeans, cargos, chinos).
- `/shoes/` - Transparent footwear layers ( Chelsea boots, sneakers).
- `/outerwear/` - Transparent outerwear layers (jackets, hoodies, coats).

## Visual Overlay Alignment

For perfect rendering alignment, ensure all transparent PNG assets share the exact same bounding dimensions (e.g. 320px x 480px) and are centered relative to the base mannequin model `body.png`.
